### Usage

```html
%markdown%
```

or 

```html
%markdown="<default_text>"%
```

### Description

A markdown text container. This is the core of an Markdown stack.

It behaves in exactly the same way as an [HTML stack](../html) but processes all markdown in the text before exporting.
